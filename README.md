# 如何使用 M3U 链接

如果仓库发布到 `https://github.com/littlefishge/IPTV`，使用的 M3U 链接是：

```text
https://raw.githubusercontent.com/littlefishge/IPTV/main/list.m3u
```

在你的 IPTV 播放器里添加这个远程播放列表 URL，即可加载 `list.m3u` 中的频道。

如果你也想查看生成时的日志文件：

```text
https://raw.githubusercontent.com/littlefishge/IPTV/main/log/info
https://raw.githubusercontent.com/littlefishge/IPTV/main/log/error
```
